# Project: vscode-gitflow

**Git Flow** VS Code extension — adds Git Flow branching workflow commands
(feature, release, bugfix, hotfix, support) as sidebar UI actions.

- **Language:** TypeScript, compiles to `./out/extension.js`
- **VS Code engine:** `>=1.64.0`
- **Repository:** https://github.com/Serhioromano/vscode-gitflow
- **Current version:** 1.4.0

## Branch model

- `main` — release tags and dependabot bumps land here
- `develop` — active development, all PRs target this branch
